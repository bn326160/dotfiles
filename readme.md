# dotfiles

My personal configuration files and development environment setup.

## Setup

Clone and apply configurations:

```bash
git clone https://github.com/brambeirens/dotfiles.git ~/.dotfiles
```

## Structure

- `nix/` - macOS and nix-darwin configurations

## Requirements

- [Nix](https://nixos.org)
- [nix-darwin](https://github.com/LnL7/nix-darwin)
